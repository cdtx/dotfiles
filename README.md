#How to install
- Clone this repository whereever wanted
- Clone the submodules
  - `git submodule init`
  - `git submodule update`
- Drive to this projects folder
- Create links
  - `ln -s vim ~/.vim`
  - `ln -s bash/bashrc_<xx> ~/.bashrc_machine`
  - `ln -s bash/bashrc ~/.bashrc`
